# Most Active Cookie

This project finds the most active cookie(s) for a given date from a cookie log file.

## Problem Statement
Given a CSV file containing cookies and timestamps, return the cookie(s) that appear the most times on a specified date.

If multiple cookies have the same highest count, all are returned (one per line).

## Requirements
- Java 8+
- Standard Java libraries only

## How to Run
./most_active_cookie.sh cookie_log.csv -d 2018-12-09
